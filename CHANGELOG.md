# Change Log

## V0.1.0

### ✨ 新功能

- **TSF 输入法引擎**：完整实现文本上屏流水线，包括组合编辑（Composition）、编辑会话（EditSession）、显示属性提供、模式切换（中/英）、按键透传与拦截，以及触控键盘布局支持
- **候选窗口渲染**：采用 GDI 实现候选框绘制，支持透明背景、字体样式和屏幕避让定位
- **TIP 生命周期管理**：支持输入法的安装、启用、禁用、卸载全流程管理，并提供独立的面板进行操作
- **设置程序**：提供完整的图形化配置界面，包含常规设置、外观设置（字体/颜色）、码表管理和关于页面
- **码表管理**：支持码表目录化浏览选择、初始化导入和预览高亮
- **外观定制**：支持选择候选框颜色、内嵌 Noto Sans SC 字体，并使用原生颜色对话框取色
- **关闭确认**：配置窗口关闭时如有未保存更改，弹出确认提示，防止误操作
- **便携模式**：自动检测便携模式优先使用本地配置目录，否则回退到 AppData
- **文件日志**：设置程序运行时自动记录日志到 log/settings.log，方便排查问题
- **Slint 候选框 UI 子系统**：实现轻量级候选框声明式 UI 定义、共享数据结构和光标定位避让算法
- **用户词典支持**：核心引擎新增用户词典开关字段，为后续自定义词组功能做准备
- **管理员权限**：设置程序嵌入 requireAdministrator 清单，确保输入法注册操作正常执行
- **候选框显示优化**：修复透明背景文字显示问题，改进无候选词时的状态同步
- **按键处理优化**：优化按键拦截与路径解析逻辑，并新增文件日志便于追踪
- **代码结构重构**：统一设置程序的脏状态管理、拆分更新逻辑、消除冗余克隆
- **注册入口整合**：输入法 DLL 的 COM 注册入口迁移至 tip_manager 统一管理
- **构建与界面优化**：构建脚本自动复制码表到输出目录；修复设置界面中不支持的 Unicode 符号
- **审查反馈修正**：根据 Spec 审查和 Code Review 反馈完善各项功能
